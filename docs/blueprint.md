# **App Name**: SrishFlow

## Core Features:

- Dynamic Header Summary: Display a real-time summary of the order status based on event details.
- Dynamic Event Details: Dynamically render event-specific input fields based on the selected event type.
- Real-time Validation and Warnings: Implement real-time form validation with non-blocking warnings for due dates and other constraints.
- Unique Order ID Generation: Generate a unique order ID upon loading the application.
- Product Configuration Cards: Display a grid layout of product catalog, and each product have a different configuration based on the defined Configuration Type.
- Billable Item Transformation: Automatically transform the product list into billable line items based on defined rules.
- Reference Pricing Auto-fill: Automatically pre-fill the Rate input fields by matching the item name against the reference pricing table.

## Style Guidelines:

- Primary color: Golden yellow (#FBB535) for primary actions, active states, and highlights.
- Background color: Light yellow (#FAF2DD), a desaturated version of the primary color, to create a warm and inviting feel.
- Accent color: Orange (#F08700) for visual contrast to indicate important action items.
- Headline font: 'Literata', a serif font for headings, to maintain a classic and professional aesthetic.
- UI font: 'Inter', a sans-serif font for UI elements, ensuring clarity and readability.
- Use clean and professional icons that align with the wedding brand aesthetic.
- Implement a responsive, dashboard-style layout with a sidebar navigation and sticky headers/footers.
- Use subtle animations for transitions and feedback to enhance the user experience.
